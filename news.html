News from Genit

Genit 2.1 is released

2013-04-02

This release adds support for Haml and Sass.

Genit 2.0 released

2013-03-29

Finally, the version 2.0 of Genit is over.

New in this release:

  • Improvement of the layout folder structure, which leads to faster compilation
  • Genit is now html5 only
  • Genit provide a web server for development
  • Config file is no longer an hidden file
  • I remove all css templates
  • Genit put an error message when the main template is malformed

Genit 1.0.1 released

2013-03-14

This is a bugfix release.

  • Fix a bug that generates a bad html5 doctype

Genit 1.0 released

2011-10-02

I am proud to announce the version 1.0 of Genit!

New in this release:

  • Genit creates a sitemap.xml
  • Genit puts nicer error messages
  • Genit puts a warning message when a here tag is used without its counterpart
  • Version is added to the .genit file

Genit 0.99 released

2011-09-25

Genit could generate an RSS feed of your news articles.
And version 1.0 is almost here!

Genit 0.9 released

2011-09-04

  • You can add news to your site with ease
  • The tag var does not exist anymore. It is replaced by the two-part tag here and what
  • Put your public files (photo, text, pdf, etc.) in the public folder
  • Put your javascript in the scripts folder
  • New --empty switch and --doctype option for the create command

Genit 0.5 released

2011-08-06

  • You can pass string variables from page to template
  • Pages can be organized into subfolders
  • Fix a bug where the cli option -v crashed

Genit 0.4.1 released

2011-07-30

Fix a bug where the closing part of auto-closing tags were forgotten.

Genit 0.4 released

2011-07-25

This is mostly an internal maintenance realease, including a better smoke test, basic user documentation and a better command-line usage.
