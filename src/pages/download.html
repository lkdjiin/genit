How to download and install Genit Download

First install Ruby and rubygem

As Genit is written in Ruby 1.9.2, you need at least the Ruby 1.9.2 interpreter to run it. You need rubygem too. Rubygem is a marvelous Ruby application that acts as an installer.
Go to www.ruby-lang.org and follow the instructions...
If you run into troubles when installing Ruby 1.9.2, you can see the book of Genit. The first chapter is about installing Ruby.

Download and install Genit

Thanks to rubygem, downloading and installing Genit is as simple as open a console and type:

  gem instal genit

All other dependencies will be downloaded and installed automatically.

Smoke test

This step is optional but performing the smoke test is a good thing to be sure the installation is well done.
Open a console window and type: 


  genit create test
  cd test
  genit compile
  genit server

Then point your browser to localhost:3000.
That's it !
