Genit documentation - compile

NAME

compile - Compile a project

SYNOPSIS

genit compile

DESCRIPTION

compile or its shortcut cc do the job of compiling your project web site. All the generated files goes to the www/ folder. Basically this command iterates over each page from the pages/ folder, injecting the content of each page into the templates/main.html file to construct your web site.

None.
