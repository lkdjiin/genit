Genit documentation - genit

NAME

genit - The main command

SYNOPSIS

genit [options] [subcommand]

genit subcommand --help

DESCRIPTION

genit is the main command of the Genit framework. It needs a subcommand to perform a useful task.

The syntax : 'genit subcommand --help' displays a list of options you can use with subcommand.

OPTIONS

-h, --help

It displays an help screen with a list of subcommands and options.

-v, --version

It displays the version number.
