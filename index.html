Open source static web framework

Quick overview

Genit is a command line framework, essentially based on conventions, to build static web site. It is designed with readability in mind. A simple example, to inject the pages in a template:

    <genit class="pages" />

May be one can say Genit is a XHTML framework...

Genit in the real world

The Genit framework is a very young one, but hey, it works!
Want a proof? You are just reading a web site made with Genit. If you are curious, you can browse the sources of this site at https://github.com/lkdjiin/genit-site.

Quick Install


    gem install genit

Genit is a Ruby application, meaning you need a Ruby interpreter to be able to run it. See installation details.

Usage


        genit create my-test-site
        cd my-test-site

(at this point edit the home page or create some more pages) 


        genit compile

Now go to the my-test-site/www/ folder and double-click the index.html to view your new site.
